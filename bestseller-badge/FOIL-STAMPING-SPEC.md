# FOIL-STAMPING PRINTER SPECIFICATION

## Cover Emblem — Production-Ready Spec

---

## FOIL TYPE

| Property | Specification |
|----------|---------------|
| **Foil Material** | Satin metallic gold foil |
| **Finish** | Non-holographic |
| **Color Bias** | Warm neutral gold (no green/yellow bias) |

### Recommended Foil Codes
- **Kurz Luxor® 385** (closest match)
- **API Foils 426 Gold** (alternative)
- **Crown Roll Leaf GP-200** (budget option)

---

## STAMP METHOD

| Parameter | Value |
|-----------|-------|
| **Process** | Blind emboss + foil stamp (single pass preferred) |
| **Emboss Depth** | 0.3–0.5 mm |
| **Secondary Detail** | Micro-bevel via die engraving |

---

## DIE REQUIREMENTS

| Requirement | Specification |
|-------------|---------------|
| **Die Material** | CNC-engraved brass die |
| **Die Type** | Dual-depth die |
| **Primary Surface** | Raised (main gold area) |
| **Secondary Detail** | Recessed (inner curves/lines) |
| **Maximum Flat Plane** | 3–4 mm (prevents "dead" areas) |

---

## EDGE FIDELITY

- ✓ Knife-sharp outer edges
- ✓ No feathering
- ✓ No rounded die edges
- ✓ Clean point termination on starburst

---

## ARTWORK REQUIREMENTS

| Format | Notes |
|--------|-------|
| **File Type** | Vector (AI, EPS, or SVG) |
| **Color Mode** | Single spot color layer |
| **Gradients** | NONE in foil art (simulated by emboss depth) |
| **Line Weight** | Minimum 0.5pt for fine details |

---

## REGISTRATION

| Tolerance | ±0.25 mm maximum |
|-----------|------------------|

---

## SUBSTRATE COMPATIBILITY

### ✓ Recommended
- Linen cover stock
- Soft-touch laminate
- Uncoated premium cover stock
- Matte laminate

### ✗ Avoid
- High-gloss lamination (kills depth illusion)
- UV coating over foil area
- Coated stocks with poor foil adhesion

---

## COLOR REFERENCE

For color matching, use these values:

| Element | Hex Code | Pantone Equivalent |
|---------|----------|-------------------|
| Base Gold | `#C9A961` | PMS 7555 C |
| Highlight | `#F5E6A8` | PMS 7401 C |
| Shadow | `#8A6F2F` | PMS 7557 C |

## QUALITY CONTROL CHECKLIST

- [ ] Sharp edge definition on all points
- [ ] Consistent foil adhesion across embossed areas
- [ ] No foil cracking at bevel transitions
- [ ] Even emboss depth throughout
- [ ] No die marks outside artwork boundary
- [ ] Proper registration on multi-color covers
